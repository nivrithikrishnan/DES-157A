Fixed Header Hero Image - Cropping / No Resizing

On this example, you will notice that the content starts below the header image and scrolls on top of it. You will also notice that if you resize the browser window, the image at the top of the page is cropped, and not resized. Note that if you make the window larger than the image is wide, you will see the image centered in the window, and it won't run all the way to the edges. If you use this technique, you could/should use media queries to load the correct sized image, depending on the size of the browser window.

This effect is created entirely in CSS. The body has a background image that is fixed to the top center of the page. Plus the body has padding on top equal to the height of the image. When the user scrolls the content, it scrolls on top of the image, off the top of the viewport, as normal.
