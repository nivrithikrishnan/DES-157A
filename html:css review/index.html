Why we use the Normalize CSS Script

The normalize script acts as a baseline style for a webpage to follow. Because different browsers implement different default stylings, the normalize script is helpful to mitigate these differences.

What is Semantic Markup

Semantic markups are HTML tags used to organize the content of a webpage in order to help search engines understand the importance of the content. It is important in order to improve accessibility, and help make content easier to read.
