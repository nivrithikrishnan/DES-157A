Inheritance in CSS

Inheritance in CSS allows for there to exist a parent-child relaionship between elements. In this scenario, child elements are able to inheret style properties from their parent allowing for the code to be cleaner, easier to read, and avoid unecessary repitition. Some examples include the color and font-family.

Specificity in CSS

Specificity allows for certain elements to be targeted and carry specific stylings. There are three main specificities used, in order of increasing specificity includes IDs(#), Classes(.), and Type selectors(p, h, etc.). The default selector is no value. Compound selectors (+, >, ~, etc.) consists of one or more single selectors and allow for a single element to have multiple conditions. Multiple selectors (,) allow you to apply the same styling to multiple elements without repitition.

ID and Class Attributes

IDs are essentially more specific than classes and can only be applied to a single element on a webpage. They are often used when an element requires some unique styling that doesn't need to be applied to other element. Classes however, are often used for groups of elements that implement a similar style. They can also be used multiple times throughout a webpage.
